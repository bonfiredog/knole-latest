ex_audio_get_falloff_ref

Gets the falloff refDistance of a sound in 3D space

Syntax

ex_audio_get_falloff_ref(soundName)
Argument Description
soundName The sound name
Returns the falloff refDistance

Description

Gets the falloff refDistance of a sound in 3D space.

Example

//Get the falloff refDistance for sound "fx_jump" and store it in variable _value
_value = ex_audio_get_falloff_ref("fx_jump");

Minimum Supported Version

exAudio 0.5.0
