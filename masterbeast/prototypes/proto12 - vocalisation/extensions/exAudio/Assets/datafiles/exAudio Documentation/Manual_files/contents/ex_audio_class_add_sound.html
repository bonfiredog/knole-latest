ex_audio_class_add_sound

Adds a sound to an audio class

Syntax

ex_audio_class_add_sound(soundName, audioClassName)
Argument Description
soundName The sound name
audioClassName The audio class name
Returns nothing

Description

Adds a sound to an audio class.

Example

//Add sound "fx_jump" to the audio class "sound_effects"
ex_audio_class_add_sound("fx_jump", "sound_effects");

Minimum Supported Version

exAudio 0.6.4
