ex_audio_get_emitter

Gets the index of the sound's emitter

Syntax

ex_audio_get_emitter(soundName)
Argument Description
soundName The sound name
Returns the sound emitter ID

Description

Returns the sound emitter ID. Naturally there is not much use for this since exAudio already takes care of creation and removal of emitters, but may come in handy for more low-level operations.

Example

//Get the emitter for sound "fx_jump" and store it in variable _value
_value = ex_audio_get_emitter("fx_jump");

Minimum Supported Version

exAudio 0.8.0
