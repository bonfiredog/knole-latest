ex_audio_class_create

Creates an audio class

Syntax

ex_audio_class_create(className)
Argument Description
className The unique class name, string
Returns the audioClassName

Description

Creates an audio class. Audio classes can be used to assign sounds for bulk manipulation. className should be a unique string otherwise the function will fail.

Example

//Create 3 new audio classes in order to add sounds to them later
ex_audio_class_create("sound_effects");
ex_audio_class_create("music");
ex_audio_class_create("voice");

Minimum Supported Version

exAudio 0.6.4
