ex_audio_get_asset_index

Gets the resource of a sound

Syntax

ex_audio_get_asset_index(soundName)
Argument Description
soundName The sound name
Returns the resource associated with the sound

Description

Returns the resource index of a sound.

Example

//Get the asset index for sound "fx_jump" and store it in variable _value
_value = ex_audio_get_asset_index("fx_jump");

Minimum Supported Version

exAudio 0.8.0
