ex_audio_listener_get_orientation_lookat_z

Gets the Z lookAt vector of the listener in the 3D space

Syntax

ex_audio_listener_get_orientation_lookat_z() Returns the listener Z lookAt vector

Description

Gets the Z lookAt vector of the listener in the 3D space.

Example

//Get the Z lookAt vector of the listener and store it in the variable _value
_value = ex_audio_listener_get_orientation_lookat_z();

Minimum Supported Version

exAudio 0.5.0
