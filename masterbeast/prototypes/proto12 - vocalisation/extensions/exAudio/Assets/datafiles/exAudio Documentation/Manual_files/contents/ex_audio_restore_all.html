ex_audio_restore_all

Restores all suspended sounds

Syntax

ex_audio_restore_all() Returns nothing

Description

Restores all suspended sounds. Paused sounds will be left alone and will not resume.

Example

//Checks if game window has gained focus and restore all suspended sounds
if (!os_is_paused()) {
    ex_audio_restore_all();
}

Minimum Supported Version

exAudio 0.5.0
