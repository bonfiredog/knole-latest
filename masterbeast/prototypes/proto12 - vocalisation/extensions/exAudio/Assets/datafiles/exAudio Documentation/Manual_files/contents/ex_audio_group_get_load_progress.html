ex_audio_group_get_load_progress

Returns the Audio Group loaded progress

Syntax

ex_audio_group_get_load_progress(audioGroup)
Argument Description
audioGroup The audio group to check
Returns the loaded percent from 0 to 100

Description

Returns the Audio Group progress. As Audio Groups are loading async, it is useful to provide some sort of loading feedback to the player. Returns the loaded percent from 0 to 100.

Example

//Get the Audio Group loading progress for "audiogroup_voices" and draw it as a text
draw_text(x, y, "Now Loading "+string( ex_audio_group_get_load_progress(audiogroup_voices) )+"%");

Minimum Supported Version

exAudio 0.8.0
