ex_audio_listener_set_position

Sets the position of the listener

Syntax

ex_audio_listener_set_position(x, y, z)
Argument Description
x The x coordinate for the listener
y The y coordinate for the listener
z The z coordinate for the listener
Returns nothing

Description

Sets the position for the listener.

Example

//Set the position for the listener
ex_audio_listener_set_position(32, 32, 0);

Minimum Supported Version

exAudio 0.5.0
