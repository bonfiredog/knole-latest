ex_audio_get_falloff_model

Gets the model on which all falloff values will be calculated

Syntax

ex_audio_get_falloff_model() Returns the falloff model

Description

Gets the model on which all falloff values will be calculated.

Example

//Get the falloff model and store it in variable _value
_value = ex_audio_get_falloff_model();

Minimum Supported Version

exAudio 0.5.2
