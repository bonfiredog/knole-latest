ex_audio_suspend_all

Suspends all sounds

Syntax

ex_audio_suspend_all() Returns nothing

Description

Suspends all sounds. Paused sounds will be left alone when restoring the sounds again with ex_audio_restore_all().

Example

//Checks if game window has lost focus and suspends all sounds
if (os_is_paused()) {
    ex_audio_suspend_all();
}

Minimum Supported Version

exAudio 0.5.0
