ex_audio_class_count_sounds

Returns the number of sounds existing in an audio class

Syntax

ex_audio_class_count_sounds(audioClassName)
Argument Description
audioClassName The audio class name
Returns the number of sounds existing in an audio class

Description

Returns the number of sounds existing in an audio class.

Example

//Get the number of sounds existing in audio class "sound_effects" and store it in the variable _value
_value = ex_audio_class_count_sounds("sound_effects");

Minimum Supported Version

exAudio 0.6.4
