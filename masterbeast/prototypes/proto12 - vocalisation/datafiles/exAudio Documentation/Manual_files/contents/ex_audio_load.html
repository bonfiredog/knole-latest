ex_audio_load

Loads and assigns a Sound resource to be used with exAudio

Syntax

ex_audio_load(soundName, soundResource, classes, priority, audioGroup)
Argument Description
soundName A unique sound name to use as identifier, string
soundResource The sound resource or an external filename string
classes (optional) A string with any existing audio classes to add the sound to, separated by space
priority (optional) The sound priority, defaults to 0
audioGroup (optional) The Audio Group that the sound belongs to, defaults to audiogroup_default
Returns the ID of the added Sound in the exAudio sound database

Description

Before you can playback or manipulate your sound with exAudio, you first need to load the sound into the system. This function registers the appointed Sound resource for use with exAudio and you can later identify the sound by the unique name that you gave it.
The name must be unique, otherwise it will fail to register.
If you pass a string as sound resource argument, then exAudio will try to load an external OGG file and create a stream from that file.
You can also pass a string with any existing Audio Classes space-separated if you wish to also add the sound to Audio Classes upon loading, otherwise leave it blank or pass a blank string "".
If your sound belongs to another audiogroup, you will need to set the audioGroup argument as well. Doing so will also automatically load that Audio Group. Because Audio Group loading is async, you can check with ex_audio_is_loaded() or ex_audio_group_is_loaded() to see if the sound and the group it belongs to is loaded yet. If you do not use Audio Groups, there is no need to check for it. ex_audio_load() should be used before using any other ex_audio functions that explicitly manipulate or playback the sound.

Example

//Load the sound "snd_fx_jump" into exAudio with the unique name "fx_jump"
ex_audio_load("fx_jump", snd_fx_jump);

//Load the external OGG sound "sndFxJump.ogg" located in working_directory into exAudio with the unique name "fx_jump"
ex_audio_load("fx_jump" "sndFxJump.ogg");

//Load the sound "snd_fx_talk1" into exAudio with the unique name "p1_lets_go" and add it to Audio Classes "sound_effects" and "voices"
ex_audio_load("p1_lets_go", snd_fx_talk1, "sound_effects voices");

//Load the sound "snd_fx_talk1" into exAudio with the unique name "p1_lets_go" and add it to Audio Classes "sound_effects" and "voices" and also load its Audio Group
ex_audio_load("p1_lets_go", snd_fx_talk1, "sound_effects voices", 0, audiogroup_voices);

Minimum Supported Version

exAudio 0.5.0
