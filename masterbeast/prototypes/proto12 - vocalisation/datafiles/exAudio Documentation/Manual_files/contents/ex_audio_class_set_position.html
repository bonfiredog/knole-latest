ex_audio_class_set_position

Sets the position of all sounds in an audio class

Syntax

ex_audio_class_set_position(audioClassName, x, y, z)
Argument Description
audioClassName The audio class name
x The x coordinate for the sound in 3D space
y The y coordinate for the sound in 3D space
z The z coordinate for the sound in 3D space
Returns nothing

Description

Sets the position of all sounds in an audio class.

Example

//Set the position for all sounds in audio class "sound_effects"
ex_audio_class_set_position("sound_effects", 55, 63, 2);

Minimum Supported Version

exAudio 0.6.4
