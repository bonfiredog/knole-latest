ex_audio_listener_set_velocity

Sets the velocity of the listener

Syntax

ex_audio_listener_set_velocity(velocityX, velocityY, velocityZ)
Argument Description
velocityX The x velocity component of the listener (default 0)
velocityY The y velocity component of the listener (default 0)
velocityZ The z velocity component of the listener (default 0)
Returns nothing

Description

Sets the velocity for the listener. This function can be used to give the listener doppler effects and simulate audio motion based on the vector that is resolved from the given relative x, y and z positions. If the listener itself is not ever going to move, or the movement is not a constant motion, you would normally not need to set these values, but, for example, if you are making a scrolling game where the player has a constant bottom to top movement and the enemies a constant top to bottom movement, you would set the listener and sound velocities (for sounds you would use ex_audio_set_velocity) to the appropriate vectors to simulate the correct doppler effect as they move past the player instance.

Example

//Check if the player instance speed is over 0 and if it is use the appropriate absolute hspeed and vspeed components to set the listener velocity
if (speed > 0) {
    ex_audio_listener_set_velocity(abs(hspeed), abs(vspeed), 0);
}

Minimum Supported Version

exAudio 0.5.0
