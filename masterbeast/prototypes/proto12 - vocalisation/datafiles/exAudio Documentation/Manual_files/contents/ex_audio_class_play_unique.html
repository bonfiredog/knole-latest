ex_audio_class_play_unique

Chooses and plays a random unique sound from an audio class

Syntax

ex_audio_class_play_unique(audioClassName)
Argument Description
audioClassName The audio class name
pitchMin The minimum random pitch value, 0 to 5
pitchMax The maximum random pitch value, 0 to 5
Returns nothing

Description

Chooses and plays a random unique sound from an audio class. The sound will be chosen randomly but in a unique manner that reduces repetitions. The function works exceptionally if the class contains a lot of sounds, preferably around 10+. Due to the nature of the function designed mainly for sound effects, there is no option to loop. Optional arguments pitchMin and pitchMax also apply a random pitch shift for more entropy.

Example

//Choose and play a random unique sound from audio class "fx_player_jumps"
ex_audio_class_play_unique("fx_player_jumps");

//Choose and play a random unique sound from audio class "fx_player_jumps" with a pitch randomized between 0.8 and 1.3
ex_audio_class_play_unique_ext("fx_player_jumps", 0.8, 1.3);

Minimum Supported Version

exAudio 0.6.4
