ex_audio_listener_set_orientation

Sets the orientation of the listener

Syntax

ex_audio_listener_set_orientation(lookatX, lookatY, lookatZ, upX, upY, upZ)
Argument Description
lookatX The x look vector (default 0)
lookatY The y look vector (default 0)
lookatZ The z look vector (default 1000)
upX The x up vector (default 0)
upY The y up vector (default 1)
upZ The z up vector (default 0)
Returns nothing

Description

Sets the orientation for the listener. The look at direction and up direction are based on the vectors that are resolved from the given relative x, y and z positions, and default to (0, 0, 1000) for the look at direction and (0, 1, 0) for the up direction. Changing the listener orientation with this function will change how sounds around the game room are perceived by the player of your game.

Example

//Use three variables to set the 3D camera projection, the 3D audio listener position and orientation too
xt = x + cos(degtorad(direction));
yt = y - sin(degtorad(direction));
zt = z - sin(degtorad(zdirection));
d3d_set_projection(x, y, z, xt, yt, zt, 0, 0, 1);
ex_audio_listener_set_position(x, y, z);
ex_audio_listener_set_orientation(xt, yt, zt, 0, 0, 1);

Minimum Supported Version

exAudio 0.5.0
