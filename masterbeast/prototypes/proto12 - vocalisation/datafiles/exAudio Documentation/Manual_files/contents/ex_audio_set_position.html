ex_audio_set_position

Sets the position of a sound in 3D space

Syntax

ex_audio_set_position(soundName, x, y, z)
Argument Description
soundName The sound name
x The x coordinate for the sound in 3D space
y The y coordinate for the sound in 3D space
z The z coordinate for the sound in 3D space
Returns nothing

Description

Sets the position of a sound in 3D space.

Example

//Set the position for sound "fx_waterfall"
ex_audio_set_position("fx_waterfall", 32, 32, 0);

Minimum Supported Version

exAudio 0.5.0
