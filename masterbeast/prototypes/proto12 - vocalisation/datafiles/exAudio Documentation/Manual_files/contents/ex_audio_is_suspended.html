ex_audio_is_suspended

Checks whether a sound is suspended

Syntax

ex_audio_is_suspended(soundName)
Argument Description
soundName The sound name
Returns true if sound is suspended, false if not

Description

Checks whether a sound is suspended. Returns true if suspended, false if not.

Example

//Check if sound "fx_jump" is suspended and if yes, restore all suspended sounds
if (ex_audio_is_suspended("fx_jump")) {
    ex_audio_restore_all();
}

Minimum Supported Version

exAudio 0.5.0
