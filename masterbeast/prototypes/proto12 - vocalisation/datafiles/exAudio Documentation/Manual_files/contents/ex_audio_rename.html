ex_audio_rename

Changes the sound's name

Syntax

ex_audio_rename(soundName, newName)
Argument Description
soundName The original sound name
newName The new name, must be unique
Returns 1 on success

Description

Changes the sound's name string. The new sound name must be unique otherwise the function will fail.

Example

//Rename the sound "fx_jump" to "fx_enemy_jump"
ex_audio_rename("fx_jump", "fx_enemy_jump");

Minimum Supported Version

exAudio 0.8.0
