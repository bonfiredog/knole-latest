ex_audio_class_exists

Checks whether an audio class exists

Syntax

ex_audio_class_exists(audioClassName)
Argument Description
audioClassName The class name
Returns true if audio class exists, false if not

Description

Checks whether an audio class exists.

Example

//Check whether "sound_effects" audio class exists, otherwise create it and add the sound
if (ex_audio_class_exists("sound_effects")) {
    ex_audio_class_add_sound("fx_jump", "sound_effects");
} else {
    ex_audio_class_create("sound_effects");
    ex_audio_class_add_sound("fx_jump", "sound_effects");
}

Minimum Supported Version

exAudio 0.6.4
