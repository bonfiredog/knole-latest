ex_audio_class_set_volume_ext

Sets the volume of all sounds in an audio class with easing

Syntax

ex_audio_class_set_volume_ext(audioClassName, value, fadeTime, easing)
Argument Description
audioClassName The audio class name
value The target value, 0 to 1
fadeTime Time to fade in/out the value in steps, 0 for instant change
easing The easing algorithm to use
Returns nothing

Description

Sets the volume of all sounds in an audio class with easing. value is the target volume from 0 = mute to 1 = full. fadeTime can be a value (measured in steps) to smoothly fade the volume in or out (depending on the target value), set it to 0 for instantaneous volume change.

Example

//Fade out the volume for all sounds in audio class "sound_effects" to 0 with easing
ex_audio_class_set_volume_ext("sound_effects", 0, 90, ease_out_quad);

Minimum Supported Version

exAudio 0.6.4
