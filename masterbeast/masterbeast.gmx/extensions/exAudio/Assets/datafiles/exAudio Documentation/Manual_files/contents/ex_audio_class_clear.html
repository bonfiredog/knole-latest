ex_audio_class_clear

Clears an audio class from all sounds

Syntax

ex_audio_class_clear(audioClassName)
Argument Description
audioClassName The class name
Returns true if cleared

Description

Clears an audio class of any sounds. Note that this doesn't actually destroy the audio class. You need to use ex_audio_class_destroy() for that.

Example

//clear audio class "sound_effects"
ex_audio_class_clear("sound_effects");

Minimum Supported Version

exAudio 0.6.4
