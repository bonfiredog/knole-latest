ex_audio_set_pitch

Sets the pitch of a sound

Syntax

ex_audio_set_pitch(soundName, value, fadeTime, easing)
Argument Description
soundName The sound name
value The target value, 0 to 5
fadeTime (optional) Time to fade the value in steps, 0 for instant change
easing (optional) The easing algorithm to use
Returns nothing

Description

Sets the pitch of a sound. Value is the target pitch from 0 = very slow to 5 = very fast. Note that values close to 0 or 5 might be a bit excessive. fadeTime can be a value (measured in steps) to smoothly fade the pitch to the target value, set it to 0 for instantaneous pitch change.

Example

//Set the pitch of sound "fx_explosion" to 1.4
ex_audio_set_pitch("fx_explosion", 1.4);

//Fade the pitch of sound "fx_explosion" to 2
ex_audio_set_pitch("fx_explosion", 2, 50);

//Fade the pitch of sound "fx_explosion" to 2 with easing
ex_audio_set_pitch("fx_explosion", 2, 90, ease_out_quad);

Minimum Supported Version

exAudio 0.6.0
