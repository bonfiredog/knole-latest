ex_audio_group_unload

Unloads an Audio Group and all its associated sounds

Syntax

ex_audio_group_unload(audioGroup)
Argument Description
audioGroup The audio group to unload
Returns 1 on success, 0 on failure

Description

Unloads an Audio Group and all its associated sounds from the exAudio sound database.

Example

//Unload the Audio Group "audiogroup_voices"
ex_audio_group_unload(audiogroup_voices);

Minimum Supported Version

exAudio 0.8.0
