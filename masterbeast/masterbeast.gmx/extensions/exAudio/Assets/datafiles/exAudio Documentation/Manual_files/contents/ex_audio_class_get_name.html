ex_audio_class_get_name

Gets the name of an audio class

Syntax

ex_audio_class_get_name(index)
Argument Description
index The index on the exAudio classes database
Returns the audio class name as a string

Description

Returns the name of an audio class by its position on the exAudio classes database. Naturally there is not much use for this since exAudio already takes care of storage and indexes, but may come in handy for more low-level operations.

Example

//Retrieve and draw all the registered audio classes
var _offset_y = 24;
for (var _i=0; _i < ex_audio_class_count(); _i++ ) {
    draw_text(window_get_width()-32, y+_offset_y, ex_audio_class_get_name(_i));
    _offset_y += 24;
}

Minimum Supported Version

exAudio 0.8.0
