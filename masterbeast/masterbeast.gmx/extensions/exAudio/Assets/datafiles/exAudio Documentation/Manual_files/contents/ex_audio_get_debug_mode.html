ex_audio_get_debug_mode

Gets whether exAudio debug messages are enabled or not

Syntax

ex_audio_get_debug_mode() Returns whether exAudio debug messages are enabled = true or not = false

Description

Gets whether exAudio debug messages are enabled or not.

Example

//Gets whether exAudio debug messages are enabled
if (ex_audio_get_debug_mode() == true) {
    draw_text(x, y, "exAudio debug mode is ON");
}

Minimum Supported Version

exAudio 0.8.0
