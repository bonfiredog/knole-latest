ex_audio_play

Plays a sound

Syntax

ex_audio_play(soundName, loop)
Argument Description
soundName The sound name
loop (optional) Whether the sound will loop. true = loop, false = play once
Returns the channelHandle or -1 if using the legacy audio system

Description

Plays a sound. The loop argument determines whether the sound will loop. true = loop, false = play once. The function returns the channelHandle of the sound playing (Added in exAudio 0.6.2), if the legacy audio system is used, it will report -1 instead.

Example

//Play the sound "fx_jump" once
ex_audio_play("fx_jump");

//Play the sound "track_stage1" looping
ex_audio_play("track_stage1", true);

Minimum Supported Version

exAudio 0.5.0
