ex_audio_set_falloff

Sets the falloff distance of a sound in 3D space

Syntax

ex_audio_set_falloff(soundName, refDistance, maxDistance, factor)
Argument Description
soundName The sound name
refDistance The falloff reference relative to the listener (clamp)
maxDistance The maximum falloff distance relative to the listener
factor The falloff factor (default 1)
Returns nothing

Description

Sets the falloff of a sound in 3D space. With this function you can set the fall-off distance for the sound. This is the distance from the listener the sound has to be before the sound will have "fallen off" to a volume of 0. The default value for this is 100, but beware of setting this to any value lower than 1 as that will make the sound played inaudible to the listener, however any other value will cause the sound to fade away the further that the emitter is from the listener.
How the sound itself is heard will depend on the falloff reference (which is the distance under which the volume for the source would normally drop by half) and the roll off factor (which affects the sound past the falloff reference distance only). The default factor is normally 1, and the effect of the different falloff values will depend on the model chosen.

Example

//Set the falloff for sound "fx_waterfall" to 300 pixels distance, starting from 100 pixels distance
ex_audio_set_falloff("fx_waterfall", 100, 300, 1);

Minimum Supported Version

exAudio 0.5.0
