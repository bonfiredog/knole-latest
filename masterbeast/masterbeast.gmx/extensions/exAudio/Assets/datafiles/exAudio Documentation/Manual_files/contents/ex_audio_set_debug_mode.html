ex_audio_set_debug_mode

Enables debug messages for exAudio

Syntax

ex_audio_set_debug_mode(enabled)
Argument Description
enabled Whether enabled = true or disabled = false
Returns nothing

Description

Enables debug messages for exAudio in the GameMaker Studio console. Turning this on may help you debug your sounds when played through exAudio and Audio Classes. By default they are turned off and should stay off when releasing your game as showing lots of debug messages may have a slight impact on performance.

Example

//Enable debug mode messages for exAudio
ex_audio_set_debug_mode(true);

Minimum Supported Version

exAudio 0.8.0
