ex_audio_get_index

Gets the index of a sound in the exAudio sound database

Syntax

ex_audio_get_index(soundName)
Argument Description
soundName The sound name
Returns the sound index

Description

Returns the index of a sound in the exAudio sound database. Naturally there is not much use for this since exAudio already takes care of storage and indexes, but may come in handy for more low-level operations.

Example

//Get the index for sound "fx_jump" and store it in variable _value
_value = ex_audio_get_index("fx_jump");

Minimum Supported Version

exAudio 0.8.0
