ex_audio_listener_get_position_y

Gets the Y position of the listener in the 3D space

Syntax

ex_audio_listener_get_position_y() Returns the listener Y position

Description

Gets the Y position of the listener in the 3D space.

Example

//Get the Y position of the listener and store it in the variable _value
_value = ex_audio_listener_get_position_y();

Minimum Supported Version

exAudio 0.5.0
