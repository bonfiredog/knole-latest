ex_audio_exists

Returns if sound exists

Syntax

ex_audio_exists(soundName)
Argument Description
soundName The sound name
Returns if sound exists

Description

Returns if the specified sound exists in the exAudio sound database and is currently loaded and ready to use with the exAudio system.

Example

//If sound "fx_jump" exists and is loaded, play it
if (ex_audio_exists("fx_jump")) {
	ex_audio_play("fx_jump", false);
}

Minimum Supported Version

exAudio 0.6.0
