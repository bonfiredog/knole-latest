ex_audio_group_is_loaded

Checks whether an Audio Group is loaded and ready

Syntax

ex_audio_group_is_loaded(audioGroup)
Argument Description
audioGroup The audio group to check
Returns true if Audio Group is loaded, false if not

Description

Checks whether an Audio Group is loaded. Returns true if loaded, false if not.

Example

//Check if Audio Group "audiogroup_voices" is loaded and if yes go to next room
if (ex_audio_group_is_loaded(audiogroup_voices)) {
    room_goto_next();
}

Minimum Supported Version

exAudio 0.8.0
