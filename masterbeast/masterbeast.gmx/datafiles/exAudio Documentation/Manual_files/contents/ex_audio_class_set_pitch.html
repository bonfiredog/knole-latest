ex_audio_class_set_pitch

Sets the pitch of all sounds in an audio class

Syntax

ex_audio_class_set_pitch(audioClassName, value, fadeTime)
Argument Description
audioClassName The audio class name
value The target value, 0 to 5
fadeTime Time to fade in/out the value in steps, 0 for instant change
Returns nothing

Description

Sets the pitch of all sounds in an audio class. value is the target pitch from 0 = very slow to 5 = very fast. Note that values close to 0 or 5 might be a bit excessive. fadeTime can be a value (measured in steps) to smoothly fade the pitch to the target value, set it to 0 for instantaneous pitch change.

Example

//Set the pitch for all sounds in audio class "sound_effects" instantly
ex_audio_class_set_pitch("sound_effects", 0.6, 0);

//Fade the pitch for all sounds in audio class "sound_effects" to 1.6
ex_audio_class_set_pitch("sound_effects", 1.6, 90);

Minimum Supported Version

exAudio 0.6.4
