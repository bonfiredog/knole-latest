ex_audio_class_resume

Resumes all the paused sounds in an audio class

Syntax

ex_audio_class_resume(audioClassName)
Argument Description
audioClassName The audio class name
Returns nothing

Description

Resumes all the paused sounds in an audio class.

Example

//Resume all the paused sounds in the audio class "sound_effects"
ex_audio_class_resume("sound_effects");

Minimum Supported Version

exAudio 0.6.4
