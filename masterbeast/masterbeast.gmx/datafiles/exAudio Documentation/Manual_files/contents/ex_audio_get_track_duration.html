ex_audio_get_track_duration

Gets the track duration of a sound

Syntax

ex_audio_get_track_duration(soundName)
Argument Description
soundName The sound name
Returns the track duration in seconds

Description

Gets the track duration of a sound in seconds, e.g. 35.12

Example

//Get the track duration for sound "fx_jump" and store it in variable _value
_value = ex_audio_get_track_duration("fx_jump");

Minimum Supported Version

exAudio 0.6.1
