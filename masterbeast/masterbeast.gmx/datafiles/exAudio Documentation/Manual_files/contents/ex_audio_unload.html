ex_audio_unload

Unloads a sound, its Audio Group and stream

Syntax

ex_audio_unload(soundName)
Argument Description
soundName The sound's name, string
Returns 1 on success

Description

When you no longer need the sound, you can remove it from exAudio with ex_audio_unload(). Unloading a sound will also check and unload the Audio Group it belongs to if no other sounds need it. Finally it will also destroy any audio streams if the sound was loaded externally from a filename. It is generally a good practice to unload any sounds that you don't need in a particular level to keep memory usage low.

Example

//Unload the sound "snd_fx_jump"
ex_audio_unload("fx_jump");

Minimum Supported Version

exAudio 0.8.0
