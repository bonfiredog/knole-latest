ex_audio_set_volume_all

Sets the volume of all sounds

Syntax

ex_audio_set_volume_all(value, fadeTime, easing)
Argument Description
value The target value, 0 to 1
fadeTime (optional) Time to fade in/out the value in steps, 0 for instant change
easing (optional) The easing algorithm to use
Returns nothing

Description

Sets the volume of all sounds. Value is the target volume from 0 = mute to 1 = full. fadeTime can be a value (measured in steps) to smoothly fade the volume in or out (depending on the target value), set it to 0 for instantaneous volume change.

Example

//Set the volume of all sounds to 0.4
ex_audio_set_volume_all(0.4);

//Fade in the volume of all sounds to 0
ex_audio_set_volume_all(1, 50);

//Fade out the volume of all sounds to 0 with easing
ex_audio_set_volume_all(0, 90, ease_out_quad);

Minimum Supported Version

exAudio 0.6.0
